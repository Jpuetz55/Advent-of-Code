## Setup
Test:

```bash
cargo check
cargo test
```

Dev build and run:

```bash
cargo run --release
cargo run --release -- --day 011 (Day 1-1)
```

Full install and run:

```bash
cargo install --path .
aoc-2023
aoc-2023 run --day 011 (Day 1-1)
```

For estimating start-up time:

```bash
aoc-2023 --empty
```
